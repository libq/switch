# the simple switch button
<h3>Step 1. Add the JitPack repository to your build file</h3>

```
    allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
```
<h3>Step 2. Add the dependency</h3>
```
dependencies {
            compile 'com.github.libq:switch:1.0.0'
	}
```
